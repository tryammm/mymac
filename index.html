Какой компьютер подходит для macOS?

Для установки macOS подойдут десктопы и ноутбуки на платформе Intel,
а так же десктопы на платформе AMD (FX, Ryzen).



На совместимости ноутбуков остановимся более детально:

Главным вопросом выступает работоспособность видеокарты.
Все современные ноутбуки с дискреной графикой работают на технологии Optimus для Nvidia и Switchable Graphics для Radeon.
Эти технологии в macOS не работают! Поэтому работать может только видео встроенное в процессор. Следовательно для установки подходят ноутбуки на процессорах со встроенным видео: Intel® HD Graphics 3000, 4000, 4400, 4600, 5000, 5500, 5600, 520, 530, 630. Так же встречаются ноутбуки в которых можно запустить дискретную графику - это редкость но такие ноутбуки существуют. В таких компьютерах заблокированная графика от интел либо ее вовсе нет в процессоре.

Краткий список процессоров на которых будет работать macOS:

поколение Core 2 Duo (запускаем дискретную графику);
первое поколение Core i3, i5, i7, при условии, что процессорное видео заблокировано (запускаем дискретную графику);

Core i3, i5, i7 Sandy Bridge
Core i3, i5, i7 Ivy Bribge
Core i3, i5, i7 Haswell
Core i3, i5, i7 Broadwell 
Core i3, i5, i7 Sky Lake 
Core i3, i5, i7 Kaby Lake 
Core i3, i5, i7, i9 Coffee Lake
Core i3, i5, i7, i9 Comet Lake
Core i3, i5, i7, i9 Ice Lake

Важным вопросом касающимся ноутбуков является работоспособность wifi модуля.

К сожалению на всех новых ноутбуках стоят модули которые в 99% не могут работать в операционной системе от Apple.

Выходов из ситуации два:

замена штатного wifi модуля на совместимый с macOS,
использование USB WIFI.
